# Learning Docker
## Cheatsheet
List Docker CLI commands
```
docker
docker container --help
```

Display Docker version and info
```
docker --version
docker version
docker info
```
 
Excecute Docker image
```
docker run hello-world
```

List Docker images
```
docker image ls
```

List Docker containers (running, all, all in quiet mode)
```
docker ps
docker ps --all
docker ps -a -q
```
